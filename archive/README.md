# Archive — the journey, kept for the story

**Read this only to recount how we got here. Nothing in `archive/` is used by active build
work** (the product in `apps/product`, the dashboard in `apps/hub`, or the canonical brand in
`brand/`). Agents working on the build should skip this folder — it exists to preserve history,
not to inform current decisions.

```
archive/
├─ explorations/
│  ├─ logos/      Rejected identity logos (02 Orbit, 03 Weave, 04 Compass, 05 Huddle)
│  │              + the oliandhue demo-tenant logo. The CHOSEN one (01 Signal) lives in brand/.
│  └─ mockups/    All pre-final exploration mockups shown to the founders.
└─ notes/         Build-history write-ups (how the Stage 1 deck & identities were constructed).
```

The presentable version of this journey is the frozen deck app at `apps/deck` (linked from the
hub's "The journey" timeline). This folder is the raw source + notes behind it.
